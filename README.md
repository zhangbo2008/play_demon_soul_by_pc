配置时候一定要右键点游戏选择config. 才是配置当前游戏的config.否则配置失败.
配置里面--gpu--white color buffer选上,其他的自己配就行.
分辨率调最低,帧率60, 把audio 禁止输出. 那么游戏更流畅!
分辨率推荐用默认的.更小会性能更高.
